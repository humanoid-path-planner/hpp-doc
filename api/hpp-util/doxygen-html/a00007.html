Detailed Description

Class that catch XML Parser events for a specific tag and build the corresponding Object.

Derive this class if you wish to extend the Parser. The event callbacks are called in the following order:

  • ObjectFactory::init after having created the object.
  • ObjectFactory::setAttribute for each attribute of the tag.
  • ObjectFactory::finishAttributes after having processed every attribute.
  • ObjectFactory::addTextChild when a child is a text element.
  • ObjectFactory::finishTags when all the children have been parsed.
  • ObjectFactory::finishFile when the file has been fully parsed.
Note
The derived class must have the following construtor
DerivedFactory (ObjectFactory* parent, const XMLElement* element) :
ObjectFactory (parent, element)
{
/*
* Keep in mind that it might be more convenient
* to build objects in an event callback, when some information
* has already been parsed.
*/
}
